## Initial Setup Commands

```bash
# Create project structure
mkdir coding-interview-platform
cd coding-interview-platform

# Initialize root package.json
npm init -y

# Create backend
mkdir backend
cd backend
npm init -y
npm install express ws cors dotenv uuid
cd ..

# Create frontend
mkdir frontend
cd frontend
npm create vite@latest . -- --template react
npm install
npm install @monaco-editor/react socket.io-client pyodide
cd ..

# Install dev dependencies at root
npm install --save-dev concurrently jest supertest
```
